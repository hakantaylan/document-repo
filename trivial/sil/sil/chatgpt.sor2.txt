1️⃣ Add a UUIDv7 library

Example (very popular):

<dependency>
  <groupId>com.github.f4b6a3</groupId>
  <artifactId>uuid-creator</artifactId>
  <version>5.3.7</version>
</dependency>

2️⃣ Create your wrapper
public final class UUIDWrapper {

    private UUIDWrapper() {}

    public static UUID generateUUID() {
        return com.github.f4b6a3.uuid.UuidCreator.getTimeOrderedEpoch();
    }
}


This produces RFC-compatible UUIDv7, sortable, fast, and safe.

3️⃣ Use it in your entity
@Entity
public class MyEntity {

    @Id
    private UUID id = UUIDWrapper.generateUUID();

    // equals & hashCode based on id ✔
}


✅ ID is never null
✅ Safe equals / hashCode
✅ No Hibernate lifecycle magic
✅ Works in constructors, tests, detached objects
