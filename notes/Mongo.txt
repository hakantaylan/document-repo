Built-in Roles
read readWrite --> DB User Roles
dbAdmin userAdmin dbOwner --> DB Admin Roles
readanyDatabase readWriteAnyDatabase userAdminAnyDatabase dbAdminAnyDatabase --> All Database Roles
clusterManager clusterMonitor hostManager clusterAdmin ---> Cluster Roles
backup restore
dbAdmin userAdmin userAdminAnyDatabase  --> Super User Roles

mongo -u hakan -p 123 --authenticationDatabase admin
